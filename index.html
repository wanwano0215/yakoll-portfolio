About

自己紹介

はじめまして、やこるです。

これまで、AIを活用した画像生成・動画生成・簡易的な動画編集をはじめ、 コミュニティ運用補佐やAI動画案件の企画進行・販売補助などに携わってきました。

制作そのものだけでなく、外注先への指示出し、スケジュール管理、進行補助、連絡対応など、 案件が円滑に進むための実務面にも幅広く対応してきました。

また、Claude Code や ChatGPT などのAIツールを活用し、 情報整理、たたき台作成、業務フロー整理なども行っています。
